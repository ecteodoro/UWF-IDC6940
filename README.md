# UWF-IDC6940
Repository for the course Capstone Projects in Data Science, University of West Florida.

### Predicting Cherry Blossom Day in Washington DC - Comparing Statistical, Machine Learning, and Deep Learning Models

This project aims to predict the peak bloom date of cherry blossoms in Washington DC using various modeling techniques, including statistical methods, machine learning algorithms, and deep learning approaches. The goal is to evaluate the performance of these models in forecasting the bloom date based on historical weather data and other relevant factors.

### Project Structure
- `data/`: Contains the datasets used for analysis and modeling.
- `notebooks/`: Jupyter notebooks for data exploration, preprocessing, and model development.
- `models/`: Scripts and files related to the different modeling techniques used.
- `results/`: Output files, including model performance metrics and visualizations.
- `README.md`: Project overview and documentation.
